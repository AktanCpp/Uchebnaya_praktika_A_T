📡 О станции и данных

Данные станции получены из базы данных системы через REST API (GET /api/stations/{id}). Информация обновляется в режиме реального времени при каждом запросе к серверу. Для добавления новой станции необходимо выполнить запрос POST /api/stations с JSON-телом, содержащим поля: name, code, location, city, latitude, longitude, parameters, description.
